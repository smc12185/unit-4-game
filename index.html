Crystal Collector

Let's find out how to play this masterpiece of a game!!!


  1. A random number will be given at the beginning of the game."Magic Number"
  2. There are four crystals below.
    • Blue
    • Green
    • Red
    • Yellow
  3. When a crystal is clicked a specific number of points will be added to your total.
  4. If you match the random number exactly, YOU WIN!!!
  5. If you go over, YOU LOSE!!. So be careful as you get closer and closer to that random number.
  6. Each crystals value will be revealed as you click it. So make sure and keep track of the ones you click!!
  7. Each time when the game starts, the game will change the values of each crystal.
